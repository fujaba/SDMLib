Yamler reads simple key value pairs in YAML syntax.

Example:

            String yaml = "" +
              "msgType: newPlayer\n" +
              "login: albert\n" +
              "colors: blue red \n";

      Yamler yamler = new Yamler();
      LinkedHashMap<String, String> map = yamler.decode(yaml);

{msgType=newPlayer, login=albert, colors=blue red}
